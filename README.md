# PDF後設資料－檔案清洗系統(論文版)
將論文內容寫入PDF的Meta內，並移除非特定論文內的雜訊

## 環境設置
### Python3
### PyPDF2
```
import PyPDF2 #導入PyPDF2套件
```
### PyMuPDF
```
pip3 install PyMuPDF #安裝PyMuPDF套件
import fitz #導入PyMuPDF套件
```
